# 基于AIoT的校园微农园智能监测系统设计

> 物联网通讯技术期末作业

## 项目简介

本项目设计了一套基于AIoT的校园微农园智能监测系统，采用**感知层-网络层-应用层**三层分层架构，实现对农作物生长环境的多维度数据采集、无线传输与智能控制。

## 系统架构

```
┌─────────────────────────────────────────┐
│            应用层 (智云平台)             │
│  图形组态 │ 物元仿真 │ 虚拟仿真 │ 智能控制│
└─────────────────────────────────────────┘
                      │
┌─────────────────────────────────────────┐
│        网络层 (WiFi + LoRa)             │
│    ESP32 │ SX1278 │ MQTT协议            │
└─────────────────────────────────────────┘
                      │
┌─────────────────────────────────────────┐
│       感知层 (传感器节点)                │
│  DHT22 │ BH1750 │ 土壤湿度 │ pH │ CO₂  │
└─────────────────────────────────────────┘
```

## 硬件清单

| 器件 | 型号 | 数量 | 用途 |
|------|------|------|------|
| 主控芯片 | ESP32-WROOM-32 | 3 | 传感器节点主控 |
| 温湿度传感器 | DHT22 | 3 | 温湿度采集 |
| 光照传感器 | BH1750 | 3 | 光照强度采集 |
| 土壤湿度传感器 | 电容式 v1.2 | 3 | 土壤湿度采集 |
| pH传感器 | pH-4502C | 3 | 土壤酸碱度采集 |
| CO₂传感器 | MH-Z19B | 3 | CO₂浓度采集 |
| LoRa模块 | SX1278 (Ra-02) | 3 | 远距离通信 |
| 继电器模块 | 5V单路 | 1 | 灌溉控制 |
| 水泵 | 12V直流水泵 | 1 | 灌溉执行 |

## 快速开始

### 1. 传感器节点部署

```bash
# 1. 安装Arduino IDE及ESP32开发板支持
# 2. 安装所需库：
#    - DHT sensor library
#    - BH1750
#    - PubSubClient (MQTT)
#    - LoRa by Sandeep Mistry

# 3. 修改WiFi和MQTT配置
# 4. 烧录sensor_read（传感器采集）.ino至ESP32
```

### 2. 数据可视化

```bash
pip install matplotlib pandas numpy
python data_display（数据可视化）.py
```
